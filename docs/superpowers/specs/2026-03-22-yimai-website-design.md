# 天津羿麦新材料科技有限公司官网设计说明

## 项目目标

为“天津羿麦新材料科技有限公司”搭建一套现代化企业官网，参考 [https://www.china-guolin.com/](https://www.china-guolin.com/) 的工业企业官网信息组织方式，同时提升整体视觉品质、首屏冲击力与移动端适配表现。

网站的核心目标：

- 清晰传达公司所处行业与主营方向
- 结构化展示五大产品体系
- 突出研发、应用、交付与服务能力
- 为后续补充真实企业资质、案例、新闻和联系方式预留位置

## 参考拆解

参考站点的可借鉴点：

- 全宽首屏与大面积工业化视觉，第一屏就建立行业认知
- 导航结构清晰，围绕“关于、产品、能力、服务、联系”展开
- 产品与基地能力分区明确，信息层级适合制造业和化工行业
- 底部包含统一联系方式与企业介绍，形成完整转化闭环

本项目不会照搬其陈旧的版式和密集信息堆叠，而是采用更现代的编排方式：

- 更强的留白与模块节奏
- 更克制的配色和玻璃感层次
- 更清晰的响应式布局
- 更强调“解决方案 + 产品体系 + 服务承诺”的现代工业企业表达

## 视觉方向

### 视觉主题

“深海蓝工艺流场”

以深海军蓝、石墨灰、低饱和青绿色为主，结合流线型光带、网格、圆弧扩散、高光边界和参数化背景，营造“化学工艺、油田流程、工业控制、稳定交付”的行业气质。

### 配色策略

- 主色：深海军蓝，体现专业、稳重、工业属性
- 辅色：青绿色，强调技术、流体、化学反应与洁净感
- 中性色：石墨灰、雾白、淡蓝灰，用于背景和信息分层
- 点缀色：低亮度金属青，仅用于按钮与重点数字，不做大面积抢色

### 版式策略

- 首屏采用全宽海报式布局，不使用传统卡片堆砌
- 每个页面保留一个主视觉焦点，避免模块之间争夺注意力
- 产品展示采用分组列表和纵深式版面，不做廉价图标九宫格
- 通过背景层、分隔线、数据数字和章节标题形成工业化秩序

## 技术方案

当前工作目录为空，项目采用静态多页面实现，便于快速交付和后续部署：

- `index.html`：首页
- `about.html`：关于我们
- `products.html`：产品中心
- `contact.html`：联系我们
- `styles/main.css`：统一视觉系统、布局、响应式与动效
- `scripts/main.js`：导航、滚动、移动端菜单、表单交互等基础行为

该方案的优点：

- 无需额外构建依赖，适合空白目录快速启动
- 易于部署到任意静态服务器
- 后续可平滑迁移到 Vite、Next.js 等工程化体系

## 交互设计

- 首屏文字与视觉层做分阶段进入动画
- 产品区块在滚动时逐步显现，增强浏览节奏
- 页头在滚动后变为更高对比的固定导航
- 移动端提供折叠菜单
- 联系页表单做前端校验与提交占位反馈

## 风险与约束

- 用户尚未提供真实地址、电话、邮箱、资质和案例，因此页面中会保留可替换文案与占位信息
- 当前项目不在 Git 仓库中，无法按技能流程执行提交动作
- 如需完全对标成熟官网，后续建议补充真实厂区、实验室、装置或应用现场图片
