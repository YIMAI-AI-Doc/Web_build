# 天津羿麦新材料官网 React 重构与品牌重设计说明

## 目标

将当前静态 `HTML/CSS/JS` 官网迁移为 `Vite + React` 结构，并同步完成以下品牌与视觉升级：

- 重做全站配色，去掉上一版偏重、偏冷硬的深色效果
- 将页头品牌简称调整为“羿麦新材料”
- 英文品牌统一为 `YIMAI NEW MATERIALS`
- 设计并落地一套新的抽象材料 / 流体风格 SVG logo
- 保留企业官网的信息逻辑，但提升现代感、克制感和高级感

## 视觉结论

### 视觉 thesis

以“矿物白工艺底板”承载“深青绿流体轨迹”，通过浅底材质感、深色文字和低饱和工业强调色，建立更适合新材料与油田化学品企业的现代官网观感。

### 内容计划

- Hero：品牌、行业定位、产品体系入口与流体型主视觉
- Support：五大产品体系概览
- Detail：应用场景、研发交付与服务逻辑
- Final CTA：联系咨询与合作入口

### 交互 thesis

- 首屏 logo 与流体图形分层进入
- 内容模块采用轻量 reveal，而不是重阴影和大面积玻璃感
- 页头在滚动中从透明过渡到矿物白底，提高阅读稳定性

## 配色方向

新配色以浅底为主，避免上一版大面积深蓝导致的压迫感。

- 背景主色：`#F4F1EA`
- 次级背景：`#ECE7DE`
- 主文字：`#22313A`
- 品牌主色：`#0F5B63`
- 辅助青灰：`#7FA6A3`
- 强调铜棕：`#C58A52`
- 分隔线：`#D7DFDC`

配色原则：

- 大面积使用矿物白和暖灰，提升舒服和耐看程度
- 以深青绿承担品牌识别和按钮强调
- 铜棕仅用于数字、小标签和 hover 细节，避免俗艳
- 弱化阴影，更多通过留白、层级和线条组织页面

## Logo 方向

新 logo 使用抽象材料 / 流体图形，不采用传统盾牌、齿轮、烧杯或过度科技的符号。

构成逻辑：

- 两条连续流体曲线交叠，形成上扬的 `Y` 型势能
- 图形整体偏几何、平衡、可缩放
- 使用深青绿为主，辅以较浅青灰形成层次
- 适配网页深浅背景、SVG 内联显示和后续印刷延展

文字策略：

- 中文：`羿麦新材料`
- 英文：`YIMAI NEW MATERIALS`

## 技术方案

采用 `Vite + React` 重构项目，使用 `React Router` 管理路由。

建议文件结构：

- `package.json`
- `vite.config.js`
- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/styles/global.css`
- `src/assets/logo-yimai.svg`
- `src/components/`
- `src/pages/Home.jsx`
- `src/pages/About.jsx`
- `src/pages/Products.jsx`
- `src/pages/Contact.jsx`

组件拆分思路：

- `SiteHeader`
- `SiteFooter`
- `BrandLogo`
- `SectionHeading`
- `Reveal`

## 页面策略

### 首页

- 保留大气首屏，但改为浅底海报式构图
- 用流体 SVG / 渐变路径作为主视觉锚点
- 产品展示改为更简洁的分段式布局，避免厚重卡片感

### 关于我们

### 产品中心

- 保持五大体系结构
- 通过分组区块、短标签和应用场景说明增强可读性

### 联系我们

- 保留联系信息和表单
- 表单区域改成更清爽、更可信的商务界面
