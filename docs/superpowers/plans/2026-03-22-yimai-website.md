# YIMAI Website Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a modern static Chinese corporate website for Tianjin Yimai New Materials Technology Co., Ltd. inspired by industrial enterprise sites and optimized for clear product communication.

**Architecture:** Use a static multi-page HTML/CSS/JS structure so the empty workspace can become a deployable website with minimal setup. Centralize the visual system and interactive behavior into shared stylesheet and script files, while each page focuses on one clear information task.

**Tech Stack:** HTML5, CSS3, vanilla JavaScript
